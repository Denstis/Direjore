# Новый GUI на CustomTkinter

## Обзор

Это полностью переписанный интерфейс для проекта "Дирижёр" с использованием библиотеки **CustomTkinter**, которая предоставляет современный внешний вид и расширенные возможности по сравнению со стандартным tkinter.

## Основные возможности

### 1. Настройки (Settings Dialog)
Полная панель настроек со всеми параметрами:

- **LM Studio**: Base URL, API Key
- **Параметры модели**: Temperature, Top P, Max Tokens, выбор моделей для Director и Worker
- **Параметры запроса**: Timeouts, Parallel Tool Calls, Stream Response
- **Лимиты**: Max Iterations, Max Plan Steps, Max Context Tokens
- **Проект**: Project Root, Log Level

### 2. Проект с уникальным именем
- При создании нового проекта генерируется уникальное имя (например, `project_a1b2c3d4`)
- Пользователь может ввести своё имя или оставить пустым для автогенерации
- Возможность переименования проекта через меню

### 3. Интерактивное окно общения
- Современный чат с цветовой кодировкой сообщений
- Поддержка спойлеров для больших сообщений от агентов
- Интерактивные кнопки для выбора опций при вопросах от системы
- Автопрокрутка к новым сообщениям

### 4. Результат работы агентов и директора
- Детальные отчёты о выполнении задач
- Списки использованных инструментов
- Созданные и изменённые файлы
- Ошибки (если есть)
- Спойлеры для скрытия/раскрытия подробностей

### 5. Панели
- **ProjectPanel**: Информация о проекте, стадии, дерево файлов
- **ChatPanel**: История сообщений, ввод текста, прогресс
- **ConfigPanel**: Табы с моделями, ролями, инструментами, памятью, логами

## Установка

```bash
pip install customtkinter>=5.2.0 darkdetect>=0.8.0
```

Или обновите requirements.txt:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
cd conductor_app
python run_new_gui.py
```

## Преимущества перед старым GUI

1. **Современный дизайн**: Тёмная/светлая тема, скруглённые углы, градиенты
2. **Спойлеры**: Возможность сворачивать большие сообщения
3. **Интерактивность**: Кнопки для быстрого выбора опций
4. **Полные настройки**: Все параметры модели и запросов в одном месте
5. **Уникальные имена проектов**: Автогенерация + переименование
6. **Лучшая производительность**: Оптимизированная архитектура

## Темы

CustomTkinter поддерживает три режима:
- System (автоматически по системе)
- Dark (тёмная)
- Light (светлая)

По умолчанию используется тёмная тема.

## Требования

- Python 3.8+
- customtkinter >= 5.2.0
- darkdetect >= 0.8.0
- tkinter (обычно входит в состав Python)
